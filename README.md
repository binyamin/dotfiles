# :card_index_dividers: Dotfiles
> My personal set of linux configuration files

- If you want to make your own dotfiles, have a look at <http://dotfiles.github.io>

## Tips

- For bash completion, place files in `~/.local/share/bash-completion/completions`. They will automatically register. Name the file according to the related executable. (e.g. for a CLI program named "foo", the file should be named "foo.bash")
